# 🍽️ Zomato Data Analysis Project
📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Zomato restaurant dataset using Python.

The goal of this project is to analyze restaurant trends, ratings distribution, country presence, online delivery availability, and popular cuisines to derive meaningful business insights.

This project demonstrates practical skills in:

Data Cleaning

Data Merging

Data Analysis using Pandas

Data Visualization using Matplotlib & Seaborn

🎯 Objective

The main objectives of this project are:

To analyze restaurant distribution across different countries and cities

To study the distribution of restaurant ratings

To identify the availability of online delivery services

To explore the most popular cuisines

To understand country-wise currency usage

To derive business insights from restaurant data

📂 Dataset Information

The project uses two datasets:

1️⃣ Zomato Dataset

Contains information about:

Restaurant Name

City

Country Code

Cuisines

Aggregate Rating

Rating Text & Rating Color

Price Range

Has Online Delivery

Currency

2️⃣ Country Code Dataset

Contains:

Country Code

Country Name

Both datasets were merged using the Country Code column to get country names.

Python

📊 Key Analysis Performed
🔹 1. Data Cleaning & Preprocessing

Checked for missing values

Verified data types

Merged country dataset with main dataset

Performed grouping and aggregation

🔹 2. Country-wise Restaurant Distribution

Identified which country has the highest number of restaurants

Observed that India has the largest share of restaurants in the dataset

🔹 3. Rating Distribution Analysis

Analyzed distribution of:

Poor

Average

Good

Very Good

Excellent ratings

Visualized rating patterns using bar plots

🔹 4. Online Delivery Analysis

Identified which countries provide online delivery

Analyzed percentage of restaurants offering online delivery

Compared delivery availability across countries

🔹 5. City-wise Restaurant Distribution

Identified top cities with maximum restaurant listings

Visualized city distribution using pie charts

🔹 6. Cuisines Analysis

Analyzed most popular cuisines

Identified frequently occurring cuisine types

🔹 7. Currency Analysis

Mapped countries with their respective currencies

Studied country-currency relationships

📈 Key Insights

🇮🇳 India dominates the dataset with the highest number of restaurants.

⭐ Majority of restaurants fall under the “Average” and “Good” rating categories.

🚚 Online delivery is available in limited countries compared to total listings.

🍜 Certain cuisines like North Indian and Chinese are highly popular.

🌍 Zomato operates across multiple countries with different currency systems.
